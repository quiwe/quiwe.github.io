Minimax API 使用指南

📅 2026-03-10 👁️ 2800 次阅读 🏷️ 教程 ⏱️ 8分钟阅读

什么是Minimax？

Minimax是国内领先的AI大模型公司，提供了强大的语言模型API服务。相比于OpenAI的API，Minimax在国内的访问速度更快，价格也更实惠。

准备工作

在使用Minimax API之前，你需要：

  1. 访问 Minimax开放平台 注册账号
  2. 创建API Key
  3. 了解API的调用方式

Python调用示例

下面是一个完整的Python调用示例：

import requests
import json

def call_minimax(prompt):
    url = "https://api.minimax.chat/v1/text/chatcompletion"
    
    headers = {
        "Authorization": "Bearer YOUR_API_KEY",
        "Content-Type": "application/json"
    }
    
    data = {
        "model": "abab6.5s-chat",
        "messages": [
            {"role": "user", "content": prompt}
        ]
    }
    
    response = requests.post(url, headers=headers, json=data)
    result = response.json()
    
    return result["choices"][0]["message"]["content"]

# 使用示例
response = call_minimax("你好，请介绍一下自己")
print(response)

常用模型

Minimax提供了多种模型：

  • abab6.5s-chat：最新版对话模型，响应速度快
  • abab6-chat：稳定版对话模型
  • abab5.5-chat：经典版本

参数说明

  • temperature：控制随机性，0-2之间，越高越随机
  • max_tokens：最大生成token数
  • top_p：核采样参数

注意事项

  • API Key要妥善保管，不要暴露在前端代码中
  • 国内访问速度快，但建议做好错误处理
  • 注意API的调用限制和费用
