AI 图像生成器

🎨 AI绘画 🖼️ 图像生成 🆕 新项目
Python Stable Diffusion OpenAI DALL-E

项目简介

一个AI驱动的图像生成工具，可以通过文字描述生成图片，支持多种风格和模型。

主要功能

  • 文字转图片（Text-to-Image）
  • 图片风格迁移
  • 图像修复和扩展（Inpainting/Outpainting）
  • 多种艺术风格支持
  • 批量生成
  • API接口支持

支持模型

  • Stable Diffusion XL - 开源强大的图像生成模型
  • DALL-E 3 - OpenAI的最新图像模型
  • Midjourney - 艺术感十足的图像生成
  • Leonardo - 专为艺术家设计

使用示例

# 命令行生成
python generate.py --prompt "a cute cat" --style anime --count 4

# API调用
import requests
response = requests.post("http://localhost:5000/generate", json={
    "prompt": "sunset over mountains",
    "style": "realistic",
    "size": "1024x1024"
})

应用场景

  • 创意设计
  • 社交媒体配图
  • 文章封面
  • 产品展示图
  • 艺术创作

技术栈
