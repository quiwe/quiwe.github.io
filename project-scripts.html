自动化脚本集合

⚡ 效率工具 🔄 持续更新
Bash Python AppleScript

项目简介

这是一个日常任务自动化脚本集合，用于提升工作效率，节省重复性操作的时间。

脚本列表

📧 邮件自动发送

定时发送日报、周报，支持自定义模板

📅 日程提醒

集成日历API，自动提醒重要会议和任务

📥 文件自动整理

自动分类下载文件夹中的文件

🌐 网页监控

监控网页变化，自动发送通知

💾 备份脚本

自动备份重要文件到云端

技术实现

  • 定时任务： Cron + launchd
  • 通知： macOS Notification Center
  • 邮件： SMTP协议
  • 文件操作： Python os/shutil模块

使用说明

每个脚本都有独立的配置文件，可以根据个人需求自定义参数。所有脚本都支持命令行运行，也可以设置为定时任务自动执行。
