GitHub Pages 个人博客搭建教程

本教程将详细介绍如何从零开始，使用 GitHub Pages 搭建一个免费的静态个人博客。即使你之前完全没有接触过 Git 和编程，也能轻松完成！

📌 前置要求

你需要准备：一个 GitHub 账号、一台电脑、网络

一、准备工作

1. 注册 GitHub 账号

  1. 打开 github.com
  2. 点击 "Sign up" 按钮
  3. 按照提示填写用户名、邮箱、密码
  4. 完成邮箱验证即可

2. 安装 Git

  1. 下载 Git：https://git-scm.com/download/win
  2. 运行安装包，全部选择默认选项即可
  3. 安装完成后，在开始菜单搜索 "Git Bash" 并打开

二、创建博客仓库

1. 创建新仓库

  1. 登录 GitHub，点击右上角 "+" → "New repository"
  2. Repository name 填写：你的用户名.github.io
  3. 例如：如果你用户名是 zhangsan，就填 zhangsan.github.io
  4. 选择 "Public"
  5. 点击 "Create repository"
💡 重要提示

仓库名必须是：用户名.github.io，这样才能通过 GitHub Pages 访问！

三、本地操作

1. 克隆仓库到本地

git clone https://github.com/你的用户名/你的用户名.github.io.git

2. 进入文件夹

cd 你的用户名.github.io

3. 创建首页文件

在文件夹中新建一个文本文件，命名为 index.html，内容如下：

<!DOCTYPE html>
<html>
<head>
    <title>我的博客</title>
</head>
<body>
    <h1>欢迎来到我的博客</h1>
    <p>这是我的第一个网页！</p>
</body>
</html>

四、上传到 GitHub

1. 添加文件到 Git

git add .

2. 提交修改

git commit -m "第一次提交"

3. 推送到 GitHub

git push -u origin main
🔑 首次推送

首次推送可能会提示登录，输入 GitHub 用户名和密码（或者 Token）即可。

五、访问你的博客

等待约 1-2 分钟，然后浏览器访问：

https://你的用户名.github.io

例如：https://zhangsan.github.io

六、后续更新博客

以后修改博客内容后，只需运行以下三命令：

git add .
git commit -m "更新内容"
git push

七、进阶：使用博客主题

如果你想用现成的主题：

  1. 访问 jekyllthemes.org
  2. 选择一个喜欢的主题
  3. 下载后解压，把文件复制到你的仓库
  4. 按照上面步骤推送即可

常见问题

Q1: 推送时提示权限错误？

需要在 GitHub 设置 → Developer settings → Personal access tokens，生成一个 Token，然后使用 Token 作为密码登录。

Q2: 网页显示 404？

检查仓库名是否正确设置为 用户名.github.io，等待几分钟后刷新。

Q3: 如何添加更多页面？

直接在文件夹中创建新的 HTML 文件，如 about.html，然后在首页添加链接即可。

🎉 恭喜你！

你已经掌握了搭建 GitHub Pages 博客的全流程！开始创作你的内容吧～
